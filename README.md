# Factorial Calculator

Це простий Golang додаток, який рахує факторіал двох чисел.

## Встановлення

Для встановлення цього додатку, склонуйте репозиторій на ваш комп'ютер:

```bash
git clone https://github.com/yeppooss/factorial_calculator.git
cd factorial_calculator
```
## Використання

Перед запуском додатку, переконайтеся, що ви налаштували вашу конфігурацію. Додаток вимагає встановлення змінної середовища CONFIG_PATH для вказання шляху до конфігураційного файлу.

```bash
export CONFIG_PATH=./config/local.yaml
```

Після того, як ви налаштували змінну середовища, ви можете запустити додаток:

```bash
go run main.go
```
